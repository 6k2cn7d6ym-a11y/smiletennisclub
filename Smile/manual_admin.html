Section 01
앱 구조 & 권한 구분

앱은 세 개의 탭으로 구성되며, 회원 관리 탭만 관리자 비밀번호가 필요해요.

①
오늘 게임 — 도착 입력, 매칭 생성, 점수 입력
일반 회원도 사용 가능 (비밀번호 불필요)
②
회원 관리 — 회원 등록·삭제·조 변경·레벨 설정·보너스 점수
🔐 관리자 비밀번호 입력 필요 · 세션 유지됨
③
랭킹 — 현재 시즌 누적 점수 순위
일반 회원도 열람 가능 (점수만 보임)
레벨(별)은 관리자 탭에서만 보여요. 일반 회원이 매칭 카드를 볼 때는 레벨 정보가 전혀 표시되지 않아요.
👥
Section 02
회원 등록 · 삭제 · 조 변경

회원 추가

1
이름 입력 후 A조 / B조 선택 → 추가 버튼
2
추가 직후 레벨(별)을 꼭 설정해주세요.
기본값은 3레벨이에요. 밸런스 매칭에 바로 반영돼요.

조 변경 & 삭제

↔
→B조 / →A조 버튼으로 즉시 이동
점수와 레벨은 그대로 유지돼요.
✕
삭제 버튼 → 해당 회원의 모든 기록 삭제
삭제 후 되돌릴 수 없어요. 신중하게!
⭐
Section 03
레벨(별) 설정 & 밸런스 매칭NEW

각 회원에게 1~5 레벨(별)을 지정하면, 매칭 생성 시 팀 밸런스를 자동으로 맞춰줘요.

★
레벨 설정 방법
회원 관리 탭 → 이름 옆 별(★★★★★) 탭
탭한 별 번호가 해당 회원의 레벨로 즉시 저장돼요.
5
레벨 기준 (참고용)
5 = 클럽 최고 고수 · 4 = 상급 · 3 = 중급 (기본값) · 2 = 초급 · 1 = 입문
레벨은 관리자 탭에서만 보여요. 경기 카드, 랭킹, 도착 목록 어디에도 표시되지 않아요.

레벨 예시

김민준
★★★★★
레벨 5 · 고수
이서연
★★★★★
레벨 4 · 상급
박지훈
★★★★★
레벨 3 · 중급 (기본)
최수아
★★★★★
레벨 2 · 초급

밸런스 매칭 로직

4명이 배정되면 가능한 팀 구성 3가지를 모두 계산해 레벨 합 차이가 가장 작은 조합을 선택해요.

예시 · 레벨 5, 4, 3, 2 배정 시
5+2 = 7
vs
4+3 = 7
✓ 차이 0 · 최적
5+3 = 8
vs
4+2 = 6
차이 2
5+4 = 9
vs
3+2 = 5
차이 4
딱 안 떨어지는 경우에도 가능한 조합 중 최소 차이를 자동 선택해요. 완벽히 균등하지 않아도 항상 최선의 조합이에요.
같은 팀 방지 로직도 보조로 함께 작동해요. 레벨 밸런스가 동점일 때 오늘 이미 같은 팀으로 뛴 적 없는 조합을 우선해요.
🎁
Section 04
보너스 점수 조정

이름 옆 숫자 입력칸에 값을 넣으면 해당 점수가 랭킹에 추가돼요. 관리자가 임의로 점수를 조정할 때 사용해요.

+
양수 입력 → 점수 추가
예: 대회 특별 점수, 개근 보너스 등
-
음수 입력도 가능 → 점수 차감
예: 미납, 페널티 등
보너스는 랭킹 점수 공식 승×3 + 무×2 + 출석×1 + 보너스에 합산돼요.
🎮
Section 05
게임 모드 관리
🏆
랭킹 모드 — 토요일 자동 적용
A조·B조 나눠서 경기 · 결과가 랭킹에 반영 · 레벨 밸런스 매칭 적용
🎮
즐겜 모드 — 일요일 자동 적용
조 구분 없이 랜덤 · 랭킹 미반영 · 레벨 밸런스 매칭은 동일하게 적용
상단 모드 배너를 탭하면 오늘 하루만 수동 변경 가능해요. 내일이 되면 자동으로 원래대로 돌아와요.
📊
Section 06
점수 & 랭킹 규칙
결과점수비고
🟢 승리+3점점수가 높은 팀
🟡 무승부+2점동점인 경우
🔴 패배+0점점수가 낮은 팀
📋 출석+1점게임 종료 시 일괄 지급
하루 3게임 랭킹 제한 — 랭킹 모드에서 하루 첫 3게임까지만 점수 합산. 4경기 이상은 게임은 진행되지만 랭킹에 미반영.
랭킹 탭에서 강등권은 A조 하위 4명, 승격권은 B조 상위 4명으로 표시돼요.
🏅
Section 07
시즌 종료 & 승강 처리

상단 오른쪽 "시즌 N 종료 →" 버튼을 눌러요.

1
A조 하위 4명 → B조로 강등
2
B조 상위 4명 → A조로 승격
3
확정 시 모든 점수 0으로 초기화, 새 시즌 시작
레벨(별)은 시즌이 바뀌어도 유지돼요.
⚠️
시즌 종료는 되돌릴 수 없어요. 반드시 랭킹을 확인하고 승강 대상자를 검토한 후 확정하세요.
🗑️
Section 08
데이터 초기화

회원 관리 탭 하단 ⚙️ 데이터 관리 섹션에 3단계 초기화 버튼이 있어요.

1
경기 기록만 초기화
회원 정보·레벨·점수는 유지 · 경기 기록만 삭제
2
랭킹만 초기화
회원·레벨 유지 · 점수·경기 기록 삭제
3
전체 초기화
회원·레벨·점수·기록 모두 삭제 · 절대 되돌릴 수 없음
⚠️
전체 초기화는 Firebase에도 반영돼요. 모든 기기에서 데이터가 사라지므로 매우 신중하게 사용하세요.
☁️
