Section 01
화면 구성 한눈에 보기

앱 상단에 탭 세 개와 모드 배너가 있어요.

①
오늘 게임 — 도착 입력, 매칭 생성, 점수 입력까지 하루 게임 전체를 여기서 진행해요.
②
회원 관리 — A조·B조 회원 등록 및 조 변경. 관리자 비밀번호가 필요해요.
③
랭킹 — 현재 시즌 누적 점수 순위를 확인해요. 이름 옆에 오늘 경기 수가 표시돼요.
상단 오른쪽 "시즌 N 종료 →" 버튼으로 시즌 마무리 및 승강 처리를 해요.
⚡
Section 02
오늘 게임 진행하기

STEP 1 · 도착 순서 입력

1
회원이 도착하는 순서대로 이름 칩을 탭하세요.
도착 목록에 번호가 붙어 추가되고, 이름 옆에 오늘 게임 수가 (N) 형태로 표시돼요.
2
도착 목록에서 ✕를 누르면 경기 종료 섹션으로 이동해요.
출석 점수는 유지되고, 현재 진행중인 경기 결과도 그대로 입력 가능해요.
3
경기 종료 섹션의 이름을 탭하면 해당 회원이 목록에서 완전히 제거돼요.
다시 참여하려면 위 도착 선택 칩에서 이름을 다시 탭하면 돼요.

STEP 2 · 매칭 생성

1
⚡ N라운드 매칭 생성 버튼을 탭해요.
2
코트 수를 선택하세요.
인원보다 적은 코트를 선택하면 대기자가 생겨요.
3
매칭 생성을 누르면 우선순위에 따라 자동으로 팀이 구성돼요.
오늘 게임 수가 적은 사람이 먼저 배정되고, 같은 팀은 최대한 안 만들어요.
🔀 다시 뽑기 — 결과를 하나도 입력하기 전이라면 라운드 헤더의 버튼으로 팀을 재구성할 수 있어요. 퇴장자와 진행중인 경기 참가자는 자동 제외돼요.
🗑 라운드 삭제 — 라운드 헤더 오른쪽 휴지통 버튼으로 해당 라운드를 삭제할 수 있어요. 입력된 점수도 자동으로 취소돼요.

STEP 3 · 점수 입력

1
경기 카드에서 팀A 점수 : 팀B 점수를 입력하고 확인을 탭해요.
이름 옆에 오늘 경기 수가 표시되고, 3경기 초과 시 (3↑)가 붙어요.
2
점수 차이에 따라 자동으로 승/무/패를 판정하고 랭킹 점수에 반영돼요.
랭킹 모드일 때만 점수가 합산돼요.
3
잘못 입력했으면 ✏️ 결과 수정을 눌러 다시 입력할 수 있어요.
🔄
Section 03
중간에 사람이 오거나 갈 때
↓
중간에 도착한 경우
위쪽 도착 선택 칩에서 이름을 탭하면 목록 맨 아래에 추가돼요. 다음 라운드부터 배정돼요.
↑
중간에 퇴장하는 경우
도착 순서 목록에서 이름 옆 ✕를 탭하면 경기 종료 섹션으로 이동해요. 이름 옆에 오늘 게임 수도 표시돼요. 진행 중인 경기 결과는 그대로 입력 가능해요.
↩
경기 종료 섹션에서 이름을 탭하면
목록에서 완전히 제거돼요. 다시 참여하려면 위 칩에서 이름을 다시 선택하세요.
🎯
Section 04
배정 우선순위

라운드 매칭 생성 시 자동으로 공정하게 배정돼요.

🥇
오늘 게임 수가 적은 사람
게임을 적게 한 사람이 먼저 배정돼요. 늦게 도착한 사람도 자동으로 먼저 배정돼요.
🥈
이전 라운드 대기자
직전 라운드에서 코트 초과로 대기한 사람
🥉
나머지 (랜덤)
이미 경기를 마친 사람들은 랜덤으로 배정
같은 팀 방지 — 오늘 이미 같은 팀으로 뛴 사람끼리는 최대한 다른 팀으로 배정해요.
경기 종료 섹션으로 이동한 사람은 다음 라운드 배정에서 자동으로 제외돼요.
📅
Section 05
게임 종료 & 기록 보기

🏁 오늘 게임 종료

1
매칭 생성 버튼 옆 🏁 종료를 누르면 오늘 게임이 마무리돼요.
2
종료 시 도착 목록과 경기 종료 목록에 있는 모든 회원에게 출석 점수가 일괄 지급돼요.
3
종료 후에는 날짜별 기록 카드가 표시돼요.
참석 인원 / 라운드 수 / 모드가 한눈에 보여요.
4
카드를 탭하면 라운드별 상세 경기 결과를 펼쳐볼 수 있어요.
다시 열기 버튼으로 종료를 취소하고 게임을 이어갈 수 있어요.
🏆
Section 06
랭킹 모드 / 즐겜 모드
🏆
랭킹 모드 — 화·목요일 자동 적용
A조·B조를 나눠서 경기하고 결과가 랭킹에 반영돼요.
🎮
즐겜 모드 — 월·수·금·토요일 자동 적용
조 구분 없이 랜덤으로 섞어서 경기해요. 점수는 랭킹에 반영되지 않아요.
화면 상단의 모드 배너를 탭하면 오늘 하루만 모드를 바꿀 수 있어요. 내일이 되면 자동으로 원래대로 돌아와요.
📊
Section 07
점수 & 랭킹 규칙
결과획득 점수비고
🟢 승리+3점점수가 높은 팀
🟡 무승부+2점동점인 경우
🔴 패배+0점점수가 낮은 팀
📋 출석+1점게임 종료 시 일괄 지급
하루 3게임 랭킹 제한 — 랭킹 모드에서 하루에 처음 3게임까지만 점수가 합산돼요. 4경기 이상은 게임은 진행되지만 랭킹 점수에는 반영되지 않아요.
랭킹 탭에서 순위 확인 가능해요. 강등권은 A조 하위 4명, 승격권은 B조 상위 4명이에요.
🔒
Section 08
회원 관리 (관리자 전용)

회원 추가·삭제·조 변경은 관리자만 가능해요. 회원 관리 탭 접근 시 비밀번호를 입력해야 해요.

🔑
관리자 비밀번호
관리자에게 별도 문의하세요
+
회원 추가 — 이름 입력 후 A조/B조를 선택하고 추가 버튼을 눌러요.
↔
조 변경 — 회원 항목의 →B조 / →A조 버튼으로 조를 바꿀 수 있어요.
+N
보너스 점수 — 이름 옆 입력칸에 숫자를 입력하면 해당 점수가 랭킹에 추가돼요.
관리자가 임의로 점수를 조정할 때 사용해요.
✕
회원 삭제 — 삭제 버튼을 누르면 해당 회원의 모든 기록이 지워져요.
⚠️ 비밀번호는 관리자만 알고 계세요. 일반 회원은 점수 입력과 게임 진행만 가능해요.
🏅
Section 09
시즌 종료 & 승강

한 시즌이 끝나면 상단 "시즌 N 종료 →" 버튼을 눌러요.

1
A조 하위 4명은 B조로 강등돼요.
2
B조 상위 4명은 A조로 승격돼요.
3
확정하면 모든 점수가 0으로 초기화되고 새 시즌이 시작돼요.
⚠️ 시즌 종료는 되돌릴 수 없어요. 내용을 꼭 확인 후 확정하세요.
📲
Section 10
홈화면에 앱처럼 추가하기

🍎아이폰 (iPhone)

1
Safari 브라우저로 앱 주소에 접속해요.
Chrome 등 다른 브라우저는 홈화면 추가가 안 될 수 있어요.
2
하단 가운데 공유 버튼 □↑을 탭해요.
3
스크롤을 내려 "홈 화면에 추가"를 탭해요.
4
이름을 확인하고 오른쪽 위 "추가"를 탭하면 완료!
홈화면에서 앱 아이콘처럼 실행되고, 전체화면으로 열려요.

🤖안드로이드 (Android)

1
Chrome 브라우저로 앱 주소에 접속해요.
2
주소창 오른쪽 ⋮ 메뉴를 탭해요.
3
"홈 화면에 추가" 또는 "앱 설치"를 탭해요.
기기나 Chrome 버전에 따라 문구가 다를 수 있어요.
4
"추가"를 탭하면 홈화면에 아이콘이 생겨요!
설치 후에는 일반 앱처럼 홈화면에서 바로 실행할 수 있어요.
